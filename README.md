# Eat da
"음식을 나누는 작은 행동, 세상을 바꾸는 큰 변화"
버려지는 잉여 음식을 사회적 약자와 연결하여 기아 종식(SDG 2)과 책임 있는 소비(SDG 12)를 실현하는 소셜 임팩트 플랫폼입니다.

# Key Features
1. 스마트 인증 및 온보딩 (Auth & Onboarding)
   - 회원가입 및 로그인: Bcrypt 기반 보안 인증.
   - 비밀번호 찾기: 본인 인증 절차를 통한 비밀번호 재설정 기능.
   - 앱 소개 온보딩: 첫 방문 유저를 위한 서비스 가이드 및 SDG 가치 전달 섹션.

2. 개인화 맞춤 설정 (User Profile)
   - 유저 타입 분류: 기부자(Donor)와 수혜자(Recipient) 타입 설정.
   - 건강 데이터 관리: 사용자의 알레르기 유무를 체크하여 데이터베이스에 저장하고, 음식 매칭 시 안전 정보를 제공합니다.

3. 메인 대시보드 및 피드 (Main & Feed)
   - 나의 활동 정보: 누적 거래 횟수(Trade Count), 북마크한 나눔, 내가 올린 포스트 관리.
   - 거리순 피드: 현재 위치 기반으로 주변의 공유 음식을 거리순으로 정렬하여 조회 가능.

4. 레시피 공유 및 커뮤니티 (Recipe & Q&A)
   - 잉여 음식 레시피: 남은 식재료를 활용한 레시피 공유 및 업로드 기능.
   - 커뮤니티 공간: 식재료 관리 팁이나 나눔 관련 질문/답변이 가능한 Q&A 게시판 운영.

5. 위치 기반 서비스 및 채팅 (Location & Communication)
   - Kakao Map API 연동: 지도를 통해 근처의 음식 공유 위치를 실시간으로 파악.
   - 1:1 실시간 채팅: 나눔 확정 및 약속 장소 조율을 위한 실시간 채팅 기능(Socket.io 활용).
